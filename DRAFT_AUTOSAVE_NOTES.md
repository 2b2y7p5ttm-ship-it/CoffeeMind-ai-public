# CM-004 — Автосохранение черновика дегустации

- Поля новой дегустации автоматически сохраняются в localStorage.
- Текущий шаг мастера также сохраняется.
- При сворачивании вкладки, уходе приложения в фон или закрытии страницы выполняется немедленное сохранение.
- После возвращения пользователь продолжает с того же места.
- После успешного сохранения дегустации черновик удаляется.
