# Sprint 1 — Mobile layout fix

## Изменения

- Восстановлена вертикальная прокрутка документа на Android.
- Разрешены вертикальные и горизонтальные touch-жесты (`pan-x pan-y`).
- Добавлена инерционная прокрутка WebKit.
- Убрано конфликтующее ограничение `overflow-x-hidden` с оболочки React.
- Нижняя навигация привязана к safe area iPhone через внутренний padding.
- Горизонтальные отступы нижней панели перенесены на единый контейнер.
- Добавлен нижний резерв контента, чтобы фиксированная навигация не перекрывала карточки.

## Проверки

- `npm run typecheck` — успешно.
- `npm run build` — успешно.

## Ручное тестирование после деплоя

1. Android Chrome: прокрутить главную страницу до последней карточки.
2. Android PWA: повторить тест после установки приложения.
