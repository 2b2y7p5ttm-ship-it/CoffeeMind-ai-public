# CM-005 — Mobile viewport restore fix

Исправлено смещение интерфейса вверх после сворачивания и повторного открытия браузера/PWA.

Изменения:
- высота мастера синхронизируется с `visualViewport.height`;
- активное поле теряет фокус перед уходом приложения в фон;
- после возврата сбрасывается ложное смещение окна;
- высота повторно пересчитывается после `pageshow`, `resize` и смены ориентации;
- черновик дегустации продолжает сохраняться.

Проверки:
- `npm run typecheck` — успешно;
- `npm run build` — успешно.
