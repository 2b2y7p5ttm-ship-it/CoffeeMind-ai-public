# CM-009 — Редактирование дегустаций и книг

## Дегустации
- Добавлена кнопка редактирования в карточке дегустации.
- Добавлен маршрут `/tasting/:id/edit`.
- Мастер дегустации открывается с уже заполненными данными.
- После сохранения обновляется существующая запись, новая копия не создаётся.
- Для редактирования используется отдельный автосохраняемый черновик.
- После сохранения пользователь возвращается в карточку дегустации.

## Книги
- В карточке книги добавлена кнопка редактирования.
- Форма открывается с текущими данными книги.
- Можно изменить статус, оценку, настроение, цитату, заметки и связанный кофе.
- После сохранения обновляется существующая запись.
- Добавлено подтверждение перед удалением книги.

## Проверки
- `npm run typecheck` — успешно.
- `npm run build` — успешно.
