# CoffeeMind AI v2.3 — Android-ready PWA

Эта версия подготовлена для Android и iPhone как публичная PWA.

## Что изменилось

- Экран установки теперь определяет Android / iPhone.
- На Android добавлена кнопка `Установить на Android`, которая использует системный PWA install prompt, если браузер его показывает.
- Добавлена ручная инструкция для Chrome Android: `⋮ → Установить приложение` или `Добавить на главный экран`.
- Manifest обновлен для Android:
  - `shortcuts` для новой дегустации, книг и Taste DNA;
  - `prefer_related_applications: false`;
  - maskable icons уже подключены.
- Service worker обновлен до `coffeemind-ai-v2.3-android-shell`.
- Главная и настройки теперь говорят “Установить на телефон”, а не только “на iPhone”.

## Как пользователи будут пользоваться

1. Пользователь открывает публичную HTTPS-ссылку в Chrome на Android.
2. Нажимает “Установить на Android” внутри приложения или открывает меню Chrome `⋮`.
3. После установки CoffeeMind появляется как отдельная иконка.
4. У каждого пользователя свой локальный журнал дегустаций и книг.

## Важно

Для настоящей установки на Android нужна публичная HTTPS-ссылка. Локальный preview из Codespaces подходит для теста, но для друзей лучше публиковать через Vercel, Netlify или GitHub Pages.
